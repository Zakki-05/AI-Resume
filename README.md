# 🚀 SmartCareer: AI-Powered Resume Builder & Analyzer

SmartCareer is a production-grade AI platform that transforms the resume building experience. Powered by Google Gemini 1.5 Flash, it helps job seekers craft professional, ATS-optimized resumes with real-time feedback and intelligent suggestions.

## ✨ Core Features

### 🛠️ AI Resume Builder
- **Multi-Step Guided Entry**: An intuitive, step-by-step form for personal info, experience, education, skills, and projects.
- **Live Professional Preview**: Instant side-by-side preview with high-end typography and layout.
- **Auto-Save Persistence**: Your data is always safe, stored automatically in local storage.

### 🧠 Gemini AI Intelligence
- **Bullet Point Optimizer**: Achievement-oriented rewriting of your work experience.
- **AI Summary Generator**: Tailored professional summaries based on your target role.
- **ATS Analyzer**: Real-time scoring and keyword detection to bypass robot filters.
- **Skill Gap Report**: Data-driven analysis of your skills vs. industry standards.

### 📄 Professional Exports
- **One-Click PDF**: Print-optimized, pixel-perfect exports ready for job applications.
- **Modern Layouts**: Professional design that works with or without profile pictures.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS, CSS Grid/Flexbox
- **Animations**: Framer Motion
- **AI Engine**: Google Gemini 1.5 Flash API
- **Icons**: Lucide React
- **Charts**: Recharts

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure AI**:
   Create a `.env` file and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```

3. **Run Locally**:
   ```bash
   npm run dev
   ```
